QUIQQER LinkedIn authentication and registration
========

This module provides a LinkedIN registration option for the `quiqqer/frontend-users` module and enables LinkedIn
authentication in QUIQQER. Users can register and sign in with their LinkedIn account.

Package Name: `quiqqer/authlinkedin`

Features
--------

* Registration via LinkedIn account (requires `quiqqer/frontend-users`)
* Authentication via LinkedIn account

Installation
------------

The Package Name is: `quiqqer/authlinkedin`
